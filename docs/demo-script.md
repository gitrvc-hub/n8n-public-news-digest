# Demo Script

## One-line intro
"This is a public-demo-safe n8n workflow that watches public sources, summarizes the most relevant updates, and delivers a clean daily digest."

## 60-second walkthrough

1. Show the homepage in `app/index.html`
2. Explain that the workflow uses public RSS feeds or APIs only
3. Open the n8n workflow JSON and point out:
   - Schedule trigger
   - Feed fetch
   - Filter / dedupe
   - AI summarization
   - Digest formatter
   - Delivery nodes
4. Show sample source items from `sample-data/feed-items.json`
5. Show a sample digest output from the project page
6. Close with business value:
   - saves research time
   - reduces noise
   - creates a repeatable daily briefing
